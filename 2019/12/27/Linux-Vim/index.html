VIM 快捷键

光标移动

  • Ctrl + u 【往前翻页】 Ctrl + f 【往前翻页 : forward】

  • gg : 第一行 GG : 最后一行

  • 0 : 行首 $ : 行尾

  • V : 选中行内容 使用 k j 来选中内容

打开文件
  • vim 1.txt 2.txt -O
